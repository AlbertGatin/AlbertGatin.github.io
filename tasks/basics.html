# Вычисление индекса массы тела

Реализовать программу для вычисления индекса массы тела

Индекс массы тела - Соответствие между массой человека и его ростом

Например, такой расчет используется в приложениях для фитнес браслетов.

Индекс массы тела (ИМТ) измеряется в кг/м² и рассчитывается по формуле: ИМТ = m/h2, где:

m масса тела в килограммах

h — рост в метрах

Пример решения 
Console.WriteLine("Введите массу в кг:");
double weight = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Введите рост в см:");
double height = Convert.ToDouble(Console.ReadLine()) / 100; // делим на 100 т.к. рост вводится в см а нужно в метрах

double imt = weight / (height * height);

Console.WriteLine($"Индекс массы вашего тела равен {imt}");

Console.WriteLine("16 и менее  Выраженный дефицит массы тела");
Console.WriteLine("16 - 18,5 Недостаточная(дефицит) масса тела");
Console.WriteLine("18,5 - 25 Норма");
Console.WriteLine("25 - 30   Избыточная масса тела(предожирение)");
Console.WriteLine("35 - 40   Ожирение второй степени");
Console.WriteLine("40 и более  Ожирение третьей степени(морбидное)");

# Кредитная анкета

Реализовать анкету для для вычисления максимальной суммы кредита

В консоль вывести заполненную анкету, и вычисленную максимальную сумму кредита

Заполненную анкету вывести в виде таблицы

Формула расчета максимальной суммы кредита (МСК)

МСК= (D х 0.3 х F )/(1 + С/100 х F/12)

где D – ежемесячный доход уже после уплаты налогов, F – срок кредита в месяцах, С – годовая процентная ставка

# Таблица истинности

Реализация 
Console.WriteLine($"x    y    !x   x&&y x||y x^y");

bool x = true;
bool y = true;

Console.WriteLine($"{x} {y} {!y} {x && y} {x || y} {x ^ y}");

# Вычисление ИМТ с условиями

Реализовать программу для вычисления индекса массы тела

Индекс массы тела - Соответствие между массой человека и его ростом

Например, такой расчет используется в приложениях для фитнес браслетов.

Индекс массы тела (ИМТ) измеряется в кг/м² и рассчитывается по формуле: ИМТ = m/h2, где:

m масса тела в килограммах

h — рост в метрах

Пример решения 
Console.WriteLine("Введите массу в кг:");
double weight = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Введите рост в см:");
double height = Convert.ToDouble(Console.ReadLine()) / 100; // делим на 100 т.к. рост вводится в см а нужно в метрах

double imt = weight / (height * height);

Console.WriteLine($"Индекс массы вашего тела равен {imt}");
if (imt <= 16)
    Console.WriteLine("16 и менее  Выраженный дефицит массы тела");
else if (imt > 16 && imt <= 18.5)
    Console.WriteLine("16 - 18,5 Недостаточная(дефицит) масса тела");
else if (imt > 18.5 && imt <= 25)
    Console.WriteLine("18,5 - 25 Норма");
else if (imt > 25 && imt <= 30)
    Console.WriteLine("25 - 30   Избыточная масса тела(предожирение)");
else if (imt > 35 && imt <= 40)
    Console.WriteLine("35 - 40   Ожирение второй степени");
else if (imt > 40)
    Console.WriteLine("40 и более  Ожирение третьей степени(морбидное)");
