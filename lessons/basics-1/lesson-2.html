# Типы данных

Задача: нам нужно узнать сумму 2х чисел

string a = "12";
string b = "24";
string sum = a + b;
Console.WriteLine(sum); //выведет 1224

Нам нужен числовой тип

int a = 12;
int b = 24;
int sum = a + b;
Console.WriteLine(sum); //36

int

int хранит целое число от -2147483648 до 2147483647 и занимает 4 байта.

Что если нужно число больше или меньше?

long num1 = -9223372036854775808;
long num2 = 9223372036854775807; 
//занимает 8 байт

int a = 12;
long b   = 24;
b = a; // так можно, так как для long выделяется больше памяти чем для int
a = b; //ошибка

Допустим мы выполняем деление 2х чисел

int a = 5;
int b = 2;
int div = a / b;
Console.WriteLine(div); /** 2 в консоль выведет только целую часть от деления хотя логично что 5/2 = 2.5, так происходит из-за того что мы выполняем операцию с целыми числами */

Нам нужен дробный тип

double a = 5d;
double b = 2d; //в данном случае можно использовать  int b = 2;
double div = a / b;
Console.WriteLine(div);  //2.5

double

double хранит число с плавающей точкой от ±5.0*10^-324 до ±1.7*10^308 и занимает 8 байта.

Что если мы характеризуем пользователя на каком то сервисе? Например хотим знать, есть платная подписка или нет?

Можно использовать

int subscribed = 0; 
int subscribed = 1;

Но тогда мы используем только 1 бит, а выделяется 4 байта

Используется тип bool имеет только 2 значения true или false

bool subscribed= true;
bool simple= false;
simple = !subscribed;

Типов данных намного больше, это же основные. Другие мы изучим позже

# Арифметические операции

# Операция вычитания двух чисел

int x = 10;
int z = x - 6; // 4

# Операция умножения двух чисел

int x = 10;
int z = x * 5; // 50

# Операция деления двух чисел

int x = 10;
int z = x / 5; // 2
 
double a = 10;
double b = 3;
double c = a / b; // 3.33333333

При делении стоит учитывать, что если оба операнда представляют целые числа, то результат также будет округляться до целого числа:

double z = 10 / 4; //результат равен 2

Несмотря на то, что результат операции в итоге помещается в переменную типа double, которая позволяет сохранить дробную часть, резултат будет целый, поскольку в самой операции участвуют два числа, которые по умолчанию рассматриваются как объекты int, то есть целые числа

Для выхода из этой ситуации необходимо определять числа или переменные, участвующие в операции, именно как тип double:

double z = 10.0 /  4.0; //результат равен 2.5
double y = 10d / 4d; //результат равен 2,5

# Операция получение остатка от целочисленного деления двух чисел

double x = 10.0;
double z = x % 4.0; //результат равен 2

# Также есть ряд унарных операций, в которых принимает участие один операнд

# Операция инкремента

Инкремент бывает префиксным: ++x - сначала значение переменной x увеличивается на 1, а потом ее значение возвращается в качестве результата операции

И также существует постфиксный инкремент: x++ - сначала значение переменной x возвращается в качестве результата операции, а затем к нему прибавляется 1

int x1 = 5;
int z1 = ++x1; // z1=6; x1=6
Console.WriteLine($"{x1} - {z1}");
 
int x2 = 5;
int z2 = x2++; // z2=5; x2=6
Console.WriteLine($"{x2} - {z2}");

# Операция декремента

Операция декремента или уменьшения значения на единицу. Также существует префиксная форма декремента --x и постфиксная x--.

int x1 = 5;
int z1 = --x1; // z1=4; x1=4
Console.WriteLine($"{x1} - {z1}");
 
int x2 = 5;
int z2 = x2--; // z2=5; x2=4
Console.WriteLine($"{x2} - {z2}");

# Приоритет операций

При выполнении сразу нескольких арифметических операций следует учитывать порядок их выполнения. Приоритет операций от наивысшего к низшему:

  1. Инкремент
  2. Декремент
  3. Умножение
  4. Деление
  5. Получение остатка
  6. Сложение
  7. Вычитание

Для изменения порядка следования операций применяются скобки.

int x = ((5 % 2) + 2)/3;
Console.WriteLine(x);

# Ввод из консоли

Что если нам нужно брать вводить данные во время выполнения программы

Например, нужно чтобы программа обращалась к нам по имени

Console.WriteLine("Как тебя зовут?");
string name = Console.ReadLine(); //команда  чтения из строки 
Console.WriteLine("Hello " + name);
Console.ReadKey();

Однако, может возникнуть вопрос, как нам быть, если, допустим, мы хотим ввести возраст в переменную типа int или другую информацию в переменные отличные от строк string

По умолчанию платформа .NET предоставляет ряд методов, которые позволяют преобразовать различные значения к типам int, double и т.д. Некоторые из этих методов:

Convert.ToInt32() //преобразует к типу int
Convert.ToDouble() //преобразует к типу double
Convert.ToDecimal() //преобразует к типу decimal
